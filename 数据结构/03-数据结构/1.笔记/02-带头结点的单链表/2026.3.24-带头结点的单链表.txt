头指针

```
头指针是只想链表第一个结点的指针
头指针具有标识作用，常用来冠以链表的名字
```

尾指针

```
尾指针是指向链表最后一个结点的指针
```

头结点

```
模型1
头结点的类型和数据结点类型一致，但是头结点不存储数据(有些地方会存储数据元素的个数)
头指针则为指向头结点的指针
作用：有了头结点，对第一个数据结点进行插入和删除操作
这种头结点也会被称为哑结点
```

```c
模型2
	专门用于管理和标识这个链表的结点，不是链表中的必要元素
	找第一个结点和最后一个结点，获取数据结点的个数会更方便
	struct node//数据结点
	{
		int data;
		struct node *next;
	};
	struct list//管理
	{
		struct node *first;//指定链表中第一个结点
		struct node *last;//指定链表中最后一个结点
		int num;//保存链表中数据结点的个数
		......
	}
```

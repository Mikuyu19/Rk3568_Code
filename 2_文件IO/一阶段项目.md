### 项目 1 《医疗科室呼号显示屏系统》

#### 1. 项目背景  
门诊科室门口壁挂屏，实时显示“当前就诊号 + 医生信息 + 等候队列”，支持语音叫号与紧急公告。

#### 2. 可以完成的功能  
| 模块           | 功能点                                             | LVGL 实现要点                                           |
| -------------- | -------------------------------------------------- | ------------------------------------------------------- |
| ① 当前叫号     | 大号字体“请 012 号 张三 到 1 诊室”                 | `lv_label` 80 px 字体，居中对齐                         |
| ② 等候队列     | 接下来 3 位候诊人                                  | `lv_table` 3 行 2 列，自动滚动                          |
| ③ 医生信息     | 头像 + 姓名 + 职称                                 | `lv_img` + `lv_label` 横向布局                          |
| ④ 叫号动画     | 新号码闪烁 3 次                                    | `lv_anim` 改变 `bg_color`                               |
| ⑤ 语音播报     | UART 发送 `“请 012 号 张三 到 1 诊室”` 给 TTS 模块 | `lv_timer` 触发后 `write(uart, buf, len)`               |
| ⑥ 紧急公告     | 全屏弹窗“系统维护中”                               | `lv_obj_set_size(lv_scr_act(), 800, 480)` + `lv_msgbox` |
| ⑦ 数据模拟     | 护士站串口指令格式：`@call,012,张三,1`             | 串口线程解析 → 更新 UI                                  |
| ⑧ 无操作回主页 | 30 s 无触摸自动关闭弹窗                            | 全局 `idle_timer` 重置                                  |

#### 3. 推荐拓展功能  
- 双队列模式：专家号 / 普通号 分栏显示  
- 语音音量加减：设置页滑动条控制 UART 指令  
- 远程升级：通过 MQTT 接收新固件 → 脚本自动 `dd` 到分区  
- 夜间模式：18:00-06:00 降低背光并切换深色样式表

###  项目 2 《智慧汽车展厅公告屏系统》

#### 1. 项目背景  
汽车销售展厅 10 寸触控桌牌，循环播放品牌视频、轮播车型海报，客户可自助预约试驾。

#### 2. 必须完成的功能  
| 模块         | 功能点                                          | LVGL 实现要点                                |
| ------------ | ----------------------------------------------- | -------------------------------------------- |
| ① 主界面轮播 | 品牌 Logo +  Slogan 每 3 s 切换                 | `lv_img` + `lv_anim` 透明度淡入淡出          |
| ② 促销卡片   | 左侧大图 + 右侧文字标题                         | `lv_obj_flex` 横向布局，自动滚动             |
| ③ 车型详情页 | 点击车型缩略图 → 进入 3 标签页：外观/参数/颜色  | `lv_tabview` + `lv_dropdown` 选颜色          |
| ④ 预约试驾   | 表单：姓名、手机号、车型、经销商、日期时间      | `lv_textarea` + 自定义软键盘 + `lv_calendar` |
| ⑤ 数据提交   | 本地校验手机号 → SD 卡保存 csv → 弹窗“提交成功” | `lv_msgbox`                                  |
| ⑥ 多语言     | 中英文切换按钮                                  | 使用 `lv_i18n` 模块，语言文件放 `/lang/`     |
| ⑦ 定时回主页 | 30 s 无操作回到轮播页                           | 全局 `idle_timer`                            |

#### 3. 推荐拓展功能  
- 视频播放：点击“品牌视频”按钮 → 调用 `ffmpeg` 解码 → `lv_canvas` 显示帧（需硬件加速）  
- 二维码生成：预约成功后显示微信客服二维码，库用 `qrcodegen`  
- 人脸识别迎宾：USB 摄像头检测到人脸 → 串口指令唤醒屏幕并播放“欢迎光临”  
- 夜间定时关机：通过 `cron` 22:00 `systemctl suspend`

###  项目 3 《多媒体自助终端一体机》

#### 1. 项目背景  
政务大厅 / 商场 21.5 寸立式终端，提供信息查询、楼层导航、广告播放、用户登录、后台管理。

#### 2. 必须完成的功能  
| 模块           | 功能点                                     | LVGL 实现要点                                |
| -------------- | ------------------------------------------ | -------------------------------------------- |
| ① 主页三分屏   | 上部视频广告、中部功能按钮、底部公告跑马灯 | `lv_obj_align` + `lv_anim` 水平滚动          |
| ② 信息查询     | 三级菜单：部门→事项→详情                   | `lv_list` → `lv_textarea` 只读               |
| ③ 楼层导航     | 简易 2D 平面图，可点击区域高亮             | `lv_canvas` 画矩形 + `lv_obj_add_event_cb`   |
| ④ 用户登录     | 手机号 + 验证码                            | `lv_textarea` + 软键盘，验证码 6 位随机      |
| ⑤ 后台管理     | 管理员输入密码 → 文件浏览 → 更新广告图     | `lv_file_explorer`（LVGL v9 自带）           |
| ⑥ 横竖屏切换   | 重力传感器中断 → 旋转 UI 90°               | `lv_disp_set_rotation(disp, LV_DISP_ROT_90)` |
| ⑦ 无操作回主页 | 60 s 无触摸 → 回到主页并清空单据缓存       | 全局 `idle_timer`                            |

#### 3. 推荐拓展功能  
- 打印机联动：查询结果页点击“打印” → UART 控制热敏打印机出票  
- NFC 读卡：读取身份证 UUID → 自动填充手机号前 7 位  
- 远程广告下发：MQTT 接收 zip → 解压到 `/mnt/ad` → 自动刷新  
- 双屏异显：HDMI 副屏播放不同广告，主屏保持交互（Linux DRM 双缓冲）

### 项目 5 《智能家居环境面板》

#### 1. 项目背景  
家用 4.3 英寸触控面板，实时显示室内温湿度、光照、气压、烟雾浓度，并能本地报警/联动空调。

#### 2. 可以完成的功能  
| 模块           | 功能点                                           | LVGL 实现要点                                                |
| -------------- | ------------------------------------------------ | ------------------------------------------------------------ |
| ① 传感器采集   | 温湿度 SHT30（I²C）                              | 每 1 s 刷新一次，使用 `lv_label_set_text_fmt()`              |
| ② 界面布局     | 顶部状态栏：Wi-Fi 图标、时间、电池               | `lv_obj_flex` 横向布局                                       |
| ③ 阈值报警     | 可本地设置温度上限、烟雾下限                     | 超阈值红色高亮卡片 + `lv_anim` 闪烁 + BEEP 引脚输出 1 kHz 方波 |
| ④ 历史曲线     | 使用 `lv_chart`，缓存最近 60 点，1 min 采样一次  | `lv_chart_set_next_value()`                                  |
| ⑤ 进入/退出    | 主页“智能家居”按钮进入；右上角 退出 图标返回主页 | `lv_btn` + `lv_event_cb`                                     |
| ⑥ 无操作回主页 | 30 s 无触摸自动关闭弹窗                          | 全局 `idle_timer` 重置                                       |

#### 3. 推荐拓展功能  
- 场景联动：点击“开启空调”按钮 → UART 向红外模块发码  
- 语音播报：UART 控制 TTS 模块，“当前温度 28 度，已为您打开空调”  
- 远程上传：通过 MQTT/Wi-Fi 把数据发到阿里云 IoT，支持手机小程序查看  
- 低功耗策略：屏幕 1 min 无操作降低背光到 30 %  
- 多房间切换：下拉框选择“客厅/卧室”，不同房间传感器地址通过 RS485 区分
